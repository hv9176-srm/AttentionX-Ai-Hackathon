##  What is this?
AttentionX AI automatically converts long-form educational videos into **short, viral reels** using AI.

##  How it works
- 🎤 Whisper → Transcribes video
- 🔊 Librosa → Extracts audio intensity
- 🤖 Gemini → Scores emotional/viral moments
- 🎞️ MoviePy → Generates vertical reel with captions

## ✨ Features
- Auto highlight detection
- AI-based emotional scoring
- Vertical (9:16) reel export

## 🧠 Use Case
Designed for:
- Educational content creators
- YouTube → Shorts / Reels conversion
